================================================================================
                           10. HOW TO USE
================================================================================

INSTALLATION:
1. Clone repository
2. Copy .env.example to .env
3. Fill in environment variables
4. Run: pnpm install
5. Run: pnpm db:generate
6. Run: pnpm db:push
7. Run: pnpm dev

FIRST TIME SETUP IN DISCORD:
1. Invite bot to server with proper permissions
2. Run /setup to configure
3. Run /panel to post ticket panel
4. Configure SellAuth API key in .env
5. Test with /status
